Instrucciones para ejecutar el programa:

1)Abrir la terminal y dirigirse a la ruta en la que se encuentra código fuente.
2)Ejecutar el programa con el comando "python3 main.py"
3)El programa desplegará dos opciones:
 - Opción 1, resolver un nonograma. Solicitará el nombre de un archivo de texto con la configuracion de renglones y columnas. Resuelve el nonograma y genera un archivo de imagen en formato png con la solución.
 - Opción 2, verificar una solución. Solicitará el nombre de la imagen con formato png y el nombre del archivo de texto con la configuración de renglones y columnas.

Nota: Para declarar regiones contiguas de las que no lo son en un mismo renglon o columna se usa un guion bajo '_'. Ejemplo: 2_3, significa que hay dos casillas contiguas, y tres contiguas separadas en un mismo renglon o columna. 

Así, también en el primer renglon del archivo de texto se declaran las columnas y en el segundo las columnas.

Solo funciona con tableros cuadrados.
